Society Management System

This project is a complete Society Management System developed using PHP, MySQL, JavaScript, AJAX, and Bootstrap. It manages members, family details, zones, cities, plans, requests, receipts, reports, and role-based dashboards for admins and users.

The system uses AJAX for dynamic data handling without page reloads, ensuring a smooth and responsive user experience. Secure session management and validations ensure proper access control and data integrity.

Project Overview

The Society Management System demonstrates real-world management functionalities including member registration, family records, payment receipts, approval workflows, reports, and zone-wise organization.

Admin features include member approvals, receipt tracking, reports, zone and city management, and request monitoring. The frontend provides a clean, responsive interface built using Bootstrap.
