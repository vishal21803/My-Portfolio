E‑Commerce Shopping Cart System

This project is a full-featured shopping cart system developed using PHP, MySQL, JavaScript, AJAX, and Bootstrap. It includes product browsing, real‑time cart updates, user login system, stock tracking, order processing, and admin-side product management.

The cart auto‑updates using AJAX without reloading the page, providing a smooth modern shopping experience. Secure session handling and validation ensure correct cart and checkout flow.

Project Overview

The Shopping Cart System demonstrates key e‑commerce functionalities including product catalog, add‑to‑cart flow, dynamic cart updates, and checkout processing. It is optimized for user experience and integrates smoothly with backend validation.

Admin features include product management, inventory tracking, and order monitoring. The frontend provides a clean, responsive layout built with Bootstrap.
