Drug Review Sentiment Analysis System

A complete machine‑learning based sentiment classification platform built to evaluate medical drug reviews. The system analyzes any user‑submitted drug review and predicts sentiment (Positive, Negative, or Neutral), assigns a score out of 5, and also calculates confidence percentage.

The backend uses a transformer model (fine‑tuned on drug review datasets) and is deployed in Flask. The frontend provides a modern UI with gradient themes, rating bars, dynamic animations, and responsive design. The sentiment logic includes probability‑based rating + keyword‑based neutral detection.

Project Overview

This sentiment analyzer system was built to analyze thousands of medical drug reviews using a fine‑tuned Transformer model. It accepts user review input, processes it through the model, and generates structured output including rating, sentiment, and confidence.

The Flask backend is optimized for fast inference, while the UI presents results using visual indicators such as rating bars and color‑coded sentiment labels. Planned upgrades include: multiple model support, multi‑drug comparison, and exporting analysis reports.
