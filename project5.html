Full Social Media Web Platform

A complete social networking system that allows users to create posts, upload images, like, comment, and interact inside a dynamic feed. The platform includes a responsive UI, post modals, smooth animations, and real-time updates using AJAX.

The system features private chat, post deletion, interactive comment threads, and a clean modern layout inspired by Instagram. All interactions are seamless without page reloads, ensuring fast performance and a richer user experience.

Project Overview

This social media platform was created to replicate the core features of modern networking apps such as Instagram. It includes an image feed, user profiles, likes, comments, chat, and post management.

The system uses a modular PHP backend combined with AJAX-based communication for instant actions. The interface is designed to be visually appealing, mobile-friendly, and easy to navigate. Future improvements include story uploads, notifications, and follow system.
