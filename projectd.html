OTT Streaming Platform

This project is a desktop-based OTT streaming clone developed using PHP, MySQL, AJAX, Bootstrap, and JavaScript. It features a clean and functional user interface, content browsing sections, and a fully working subscription-based autopay system that automatically renews plans without manual input.

Although the platform currently uses a default video file instead of real movies, it efficiently simulates the structure and functionality of a true OTT service. Built for PC and laptop screens, it highlights responsive design, secure data handling, and smooth user experience for streaming prototypes.

Project Overview

The OTT Streaming Platform Clone is designed to demonstrate how online streaming services function. It allows users to explore a modern interface, subscribe to different plans through an automated payment system, and experience smooth dashboard navigation built using PHP and MySQL.

This version currently supports only PC and laptop screens and plays a default sample video for testing. Despite limited content, the system efficiently handles user authentication, plan verification, and billing flow, showcasing the essential framework of a real OTT application.
